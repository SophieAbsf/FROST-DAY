# Installation

## Stable release

To install FROST-DAY, run this command in your terminal:

```sh
uv add frost_day
```

Or if you prefer to use `pip`:

```sh
pip install frost_day
```

## From source

The source files for FROST-DAY can be downloaded from the [Github repo](https://github.com/SophieAbsf/frost_day).

You can either clone the public repository:

```sh
git clone https://github.com/SophieAbsf/frost_day
```

Or download the [tarball](https://github.com/SophieAbsf/frost_day/tarball/main):

```sh
curl -OJL https://github.com/SophieAbsf/frost_day/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd frost_day
uv sync
```
